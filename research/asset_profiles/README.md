# 🪙 Asset Profiles & Targeted Sentiment Knowledge Base

โฟลเดอร์นี้ใช้เป็น **"สมองส่วนหน้า" (Fundamental & Sentiment Context)** สำหรับ AI เพื่อใช้ประกอบการตัดสินใจเชืงปริมาณ (Quant) โดยมุ่งเน้นการวิเคราะห์ที่มีระดับความเฉพาะเจาะจงสูงต่อแต่ละสินทรัพย์ (Asset-Specific)

---

## 🎯 แนวคิดหลัก (Core Concept)
แทนที่จะกวาดข้อมูลข่าวและ Social Media ทั้งตลาด (ซึ่งมีอัตรา Noise สูง) เราจะสร้าง **"Profile"** ของแต่ละเหรียญ และกำหนดแหล่งข้อมูล (Sources) ที่ส่งผลกระทบต่อเหรียญนั้นๆ อย่างแท้จริง

### ข้อมูลในแต่ละ Profile (โครงสร้างไฟล์)
แต่ละไฟล์ในโฟลเดอร์นี้ควรรหัสนามสกุล `.md` ตามชื่อเหรียญ (เช่น `BTC.md`, `DOGE.md`, `GALA.md`) โดยมีโครงสร้างดังนี้:

1. **Asset Identity:** เหรียญนี้คืออะไร? อยู่ใน Sector ไหน? (เช่น L1, GameFi, AI, Meme)
2. **Key Drivers:** ปัจจัยหนุนหลักคืออะไร? (เช่น อัตราการเบิร์นเหรียญ, จำนวนผู้เล่นเกม Active)
3. **KOLs / Influencers:** บัญชี Twitter หรือแหล่งข่าวไหนที่มีผลกับกราฟของเหรียญนี้โดยตรง (Targeted Sentiment Sources)
4. **Sentiment Rules:** กฎเกณฑ์ที่ AI ควรมองหา (เช่น ถ้า Elon Musk แซวเรื่องน้องหมา = Positive Sentiment Score +10 สำหรับ Meme coins)
